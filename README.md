# Farm Farmer

An incremental RPG for Android — build a roster of heroes, push them through floor after floor of
combat, and train a deep bench of gathering/crafting skills to keep them equipped.

**Status:** early development. No public build yet. This repo currently holds design docs and
economy simulators from Phase 1; the Unity project scaffold is in progress (Phase 2).

## Repo layout

```
docs/    — game design, art direction, and economy documentation
sim/     — Python simulators used to tune economy/skill constants
Assets/  — Unity project (once committed)
```

See `CLAUDE.md` for full project context, locked design decisions, and conventions.
